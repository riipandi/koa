# Basic Koa Project

A basic Koa programming language project example.

## Getting Started

```bash
# Run the project
koa run

# Build the project
koa build

# Build in release mode
koa build --mode release
```

## Project Structure

```
.
├── Koa.toml      # Project configuration
├── README.md     # This file
└── src/
    └── main.koa  # Entry point
```
